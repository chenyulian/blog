📁 项目 Selected Projects

# Drawing

Drawing 是一款基于canvas的Web绘图应用，支持压感。

# Xela UI

Xela UI 是一款小型UI组件库，基于 Vue 3和 TypeScript。这个UI库是以学习 Vue 3为目的编写的，仅供学习使用。

# Vue ToDo

Vue ToDo 是一款基于 Vue和 TypeScript的 TODO类Web app，是我学习Vue编写的第一个应用。由于之前主语言是Java，对面向对象风格更适应，本项目的自定义组件均采用Class style编写。此外，使用了ElementUI的部分组件简化开发，Vuex管理数据，localStorage存储数据，ECharts做数据统计图表。
