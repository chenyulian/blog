作用域

var只有两种作用域：全局作用域和函数作用域。

在let和const出现之前，程序员都是用立即执行函数（IIFE）来模拟块级作用域的。

let和const还有块级作用域——包括代码块{}、if-else、for语句。

总结

var、let、const的区别是：

  1. var声明的变量，可以在其声明语言前被使用；let和const需要先声明再使用。

  2. var可以重复声明，let和const不能重复声明；

  3. var只有全局作用域和函数作用域，let和const还有块级作用域；

  4. var 在全局作用域下声明变量会导致变量挂载在 window 上，其他两者不会

  5. let声明的变量复制后值能改变，const声明的变量赋值后不能再改。
